# 주류 유통 소매기업 ERP 데이터 기반 재고 운영 효율화

주류 유통 소매기업의 분산된 가상 ERP 데이터(매출, 재고, 매입 등)를 결합·분석해, 핵심 운영 비효율을 개선하기 위한 최적화된 재고 관리 방안을 도출한 프로젝트입니다.

## 개요

- **소속**: 서강대학교 경영데이터사이언스 학회 INSIGHT
- **대회/기간**: 2025년 2학기 1차 인사이콘(학회 내부 경진대회), 2025.09~2025.10 진행
- **목표**: 주류 유통 소매기업의 분산된 가상 ERP 데이터(매출, 재고, 매입 등)를 결합 및 분석하여, 핵심 운영 비효율을 개선하기 위한 최적화된 재고 관리 방안 도출
- **팀**: 뽀삐삐뽀 (버디조 2조) — 이종현, 김재성, 권도영, 김현서, 안가은

## 담당 역할

팀원 (핵심 전략 도출) — 회귀분석 기반 수요예측(`strategy/회귀분석_최종.ipynb`)으로 예상 수요를 산출하고, 이를 바탕으로 과잉 재고 문제를 규명해 ROP(재주문점) 발주 전략을 설계·시뮬레이션. 해당 전략으로 과잉 재고 50% 감축 효과를 정량적으로 검증

## 주요 내용 및 성과

- **ERP 데이터 병합 및 전처리**: 매출, 재고, 매입 등 기업의 분산된 가상 ERP 테이블을 고유 키(Key) 기준으로 결합(Join)·정제하여 분석에 최적화된 전처리 데이터셋 구축
- **비즈니스 비효율 원인 규명**: 선입선출(FIFO) 기반 재고 소진 흐름, 매입 리드타임, 예상 수요 등을 종합 분석해 핵심 운영 비효율 원인이 '과잉 재고'에 있음을 규명
- **데이터 기반 솔루션 제안 및 검증**: 데이터 기반 재주문점(ROP) 자동 주문 시스템 도입을 해결책으로 제안하여, 과잉 재고를 50% 감축하는 정량적 시뮬레이션 성과 도출

복잡한 ERP 시스템의 데이터 구조를 파악하고 비즈니스 비효율을 진단해, 실질적인 프로세스 개선 솔루션을 제안하는 실무적 문제 해결력을 이 프로젝트에서 보여줬습니다.

## 파이프라인

1. **전처리** (`preprocess/`)
   - `preprocess.ipynb` — 매출·재고·매입 등 분산된 ERP 테이블 결합 및 정제
   - `fifo.ipynb` — 선입선출(FIFO) 기준 재고 소진 흐름 계산
2. **전략 도출** (`strategy/`)
   - `store_id_매출추이.ipynb` — 매장별 매출 추이 분석
   - `brand_id_군집화.ipynb` — 브랜드 군집화(KMeans/DBSCAN)
   - `cluster0_ABC.ipynb` — 군집별 ABC 분석
   - `회귀분석_최종.ipynb` — 회귀분석 기반 수요예측 및 ROP 발주 전략 시뮬레이션 (담당 파트)
